# GeoDefesa

Aplicação mobile desenvolvida com objetivo de prever e mostrar todas as áreas de risco de Campina Grande-PB.

##### Detalhes de desenvolvimento
* Linguagem: java
* Api: arcgis 3.x
* Ide: android studio
